
### MazeCore — Програма генерації лабіринтів та пошуку виходу

**MazeCore** — це десктопний застосунок на базі .NET 8 та WPF, призначений для процедурної генерації ідеальних лабіринтів та візуалізації алгоритмів пошуку найкоротшого шляху. Проєкт розроблений у рамках навчальної практики за спеціальністю «Інженерія програмного забезпечення».

## 🚀 Основні можливості

- **Генерація лабіринтів**: Використання алгоритму **Recursive Backtracker** (DFS) на основі стеку для створення складних і заплутаних структур.
- **Пошук виходу**: Реалізація алгоритму **BFS (Пошук у ширину)** для знаходження гарантовано найкоротшого шляху з візуалізацією процесу в реальному часі.
- **Система авторизації**: Розмежування прав доступу для ролей **Admin** та **User**. Паролі захищені за допомогою криптографічного хешування **BCrypt**.
- **Керування даними (CRUD)**: Повний цикл створення, читання, оновлення назв та видалення лабіринтів.
- **Розумний пошук**: Використання алгоритму **Левенштейна (Fuzzy Search)** для пошуку лабіринтів у базі навіть при наявності помилок у назві.
- **Власне сортування**: Реалізація **QuickSort** для впорядкування бази лабіринтів за складністю та назвою.
- **Персоналізація**: Підтримка світлої/темної тем та локалізації (українська/англійська мови).
- **Логування**: Фіксація всіх ключових дій користувачів у системному файлі `logs.json`.

## 🛠 Технологічний стек

- **Мова**: C# 12

- **Інтерфейс**: WPF (XAML)
- **Формат даних**: JSON (System.Text.Json)
- **Бібліотеки**: BCrypt.Net-Next

## 📁 Структура проєкту

- `/Models` — Сутності даних (User, Maze, SolveRecord).
- `/Views` — WPF сторінки та вікна.
- `/Services` — Бізнес-логіка (Генератор, PathFinder, AuthService).
- `/Common` — Універсальні інструменти (JSON провайдер, Сортування, FuzzySearch).
- `/Resources` — Стилі, теми та файли локалізації.
- `/Data` — Локальна база даних (автоматично створюється при запуску).
